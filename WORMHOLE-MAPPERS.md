# Wormhole Mapping Tools (2026 Update)

As of March 2026, the wormhole mapping landscape has shifted significantly following the retirement of major public hosts.

## Recommended Tools

### 1. GalaxyFinder
The rising star for modern WH mapping. Cloud-native, secure, and highly scalable.
* **URL:** [https://wormholes.new-eden.io/](https://wormholes.new-eden.io/)
* **Status:** Active, frequent updates (v5.0+).
* **Vibe:** Polished, modern UI, very stable.

### 2. Pathfinder (Community Resurrections)
The original `lost-in-w.space` is discontinued, but the code lives on in community-hosted instances.
* **LinkNet Pathfinder:** [https://pathfinder.eve-linknet.com/](https://pathfinder.eve-linknet.com/)
* **Self-Hosting:** Many corps now use the Goryn Clade fork: [https://github.com/goryn-clade/pathfinder](https://github.com/goryn-clade/pathfinder)
* **Status:** Active via community hosts.

### 3. Tripwire
The "old faithful" of mapping tools. 
* **URL:** [https://tripwire.eve-apps.com/](https://tripwire.eve-apps.com/)
* **Status:** Active.
* **Vibe:** Classic/Legacy feel, but fast and reliable.

### 4. Vippy
Used by high-end wormhole entities for tactical fleet tracking.
* **URL:** [https://eve-vippy.com/](https://eve-vippy.com/)
* **Status:** Active.
* **Vibe:** Tactical, fleet-focused, powerful but niche.
